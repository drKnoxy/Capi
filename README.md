# Capi 🦫

A [PICO-8](https://www.lexaloffle.com/pico-8.php) game about a capybara trying to earn money to upgrade his hotspring.

## How to Play

Open `capi.p8` in PICO-8 to play.

### Controls

| Button | Action |
|--------|--------|
| Arrow keys | Move the capybara |
| **Z** | Interact (open upgrade menu near hotspring) |
| **X** / Left arrow | Close menu |

### Goal

Collect coins scattered around the map and earn money by working at the bamboo patch. Spend your coins to upgrade your hotspring from a tiny puddle all the way to a paradise!

### Hotspring Upgrades

| Level | Name | Cost |
|-------|------|------|
| 1 | Tiny Puddle | Free (starting) |
| 2 | Small Pool | 50 coins |
| 3 | Warm Bath | 150 coins |
| 4 | Cozy Spring | 300 coins |
| 5 | Paradise | 600 coins |

### Earning Coins

- 🪙 **Collect coins** — Walk over the glittering coins scattered across the map
- 🎋 **Work** — Stand near the bamboo work area in the bottom-right to earn +3 coins every 5 seconds

### Features

- Title screen with animated background
- Capybara character with walking animation and directional facing
- Hotspring that visually grows and gains decorations with each upgrade
- Steam particles that increase with hotspring level
- Coin progress bar in the upgrade menu
- HUD showing current coin count and hotspring level
